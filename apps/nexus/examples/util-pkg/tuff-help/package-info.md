# help

## Package Overview
The `help` package is a utility library that provides a file tree generator for the Talex Touch project. It helps visualize the directory structure of the project.

## Key Features
- File tree generation (tree-generator.ts)

## Dependencies
The package uses the following modules:
- node:fs - Node.js file system module
- node:path - Node.js path module
- fs-extra - Extended file system module

## Usage
The main function `genFileTree` can be used to generate a visual representation of a directory structure with configurable depth and ignore patterns.
