# TUFF Help Utilities

## Introduction

The TUFF Help Utilities package provides helpful tools for the Talex Touch project. It includes utilities that assist with project visualization, directory structure analysis, and other helpful functions that support development and understanding of the project.

## Features

- **File Tree Generator**: A utility to generate and display directory structures in a tree format
- **Project Visualization**: Tools for visualizing the project's file organization
- **Directory Analysis**: Functions to analyze and understand the project's directory structure

## Usage

The main utility in this package is the `genFileTree` function, which can be used to generate a visual representation of a directory structure with configurable depth and ignore patterns.
