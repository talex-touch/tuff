# test

## Package Overview
The `test` package is a testing library for the Talex Touch project. It provides various test utilities and modules that are used to test different parts of the application.

## Key Features
- AISDK tests (aisdk)
- Common tests (common)
- Core Box tests (core-box)
- Download tests (download)

## Dependencies
- @talex-touch/utils (workspace:^1.0.0) - Direct dependency
- ts-node (^10.9.2) - Direct dependency
- vitest (^3.2.4) - Direct dependency

## Usage
This package contains test files and utilities that can be used to test different parts of the Talex Touch application. It uses vitest as the testing framework.
