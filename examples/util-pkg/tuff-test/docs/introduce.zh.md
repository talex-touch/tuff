# TUFF 测试框架

## 介绍

TUFF 测试框架是为 Talex Touch 生态系统设计的综合测试解决方案。它提供各种测试实用程序和模块，确保应用程序不同部分的质量和可靠性。

## 功能

- **AISDK 测试**: 用于 AI 相关功能的测试组件
- **通用测试**: 共享测试实用程序和通用测试用例
- **Core Box 测试**: Core Box 功能的特定测试
- **下载测试**: 用于下载相关功能的测试实用程序
- **集成测试**: 跨模块的综合集成测试

## 用法

测试框架使用 Vitest 作为主要测试框架，并为测试 Talex Touch 应用程序的不同组件提供结构化的方法。它包括用于设置测试环境、运行测试和生成覆盖率报告的实用程序。
