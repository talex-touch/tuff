# @talex-touch/utils

## Package Overview
The `@talex-touch/utils` package is a utility library for the Talex Touch project. It provides various utility functions and modules that are commonly used across different parts of the application.

## Key Features
- Base utilities (base)
- Channel communication utilities (channel)
- Common utilities (common)
- Core Box utilities (core-box)
- Event bus implementation (eventbus)
- Intelligence utilities (intelligence)
- Plugin utilities (plugin)
- Preload utilities (preload)
- Type definitions (types, types/download, types/icon, types/update)

## Dependencies
- electron (^37.2.4) - Peer dependency
- vue (^3.5.15) - Peer dependency
- @vueuse/core (^13.6.0) - Direct dependency
- path-browserify (^1.0.1) - Direct dependency

## Usage
This package is designed to be a shared utility library that can be imported and used across different parts of the Talex Touch application. It contains commonly used functions, type definitions, and utility classes.
