# 组件

TUFF UI 提供了丰富的组件库，涵盖了现代 Web 应用开发的各种需求。

## 基础组件

- **[TuffButton 按钮](./button.md)** - 触发操作的基础组件
- **[TuffFlatButton 扁平按钮](./flat-button.md)** - 扁平风格按钮

## 表单组件

- **[TuffInput 输入框](./input.md)** - 表单输入组件
- **[TuffSwitch 开关](./switch.md)** - 开关切换组件
- **[TuffCheckbox 复选框](./checkbox.md)** - 复选框组件
- **[TuffSelect 选择器](./select.md)** - 下拉选择组件

## 反馈组件

- **[TuffProgress 进度条](./progress.md)** - 进度展示组件

## 快速开始

```bash
npm install @talex-touch/tuff-ui
```

```vue
<template>
  <TuffButton type="primary">Hello TUFF UI</TuffButton>
</template>

<script setup>
import { TuffButton } from '@talex-touch/tuff-ui'
</script>
```
