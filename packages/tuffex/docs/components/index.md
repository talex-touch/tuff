TuffEx UI 提供了丰富的组件库，涵盖了现代 Web 应用开发的各种需求。

- **[TxButton 按钮](./button.md)** - 按钮组件

- **[TxAvatar 头像](./avatar.md)** - 头像组件
- **[TxSwitch 开关](./switch.md)** - 开关组件
- **[TxSelect 选择器](./select.md)** - 下拉选择组件
- **[TxSlider 滑块](./slider.md)** - 区间取值滑块
- **[TxTooltip 文字提示](./tooltip.md)** - Hover / Focus 提示
- **[TxPopover 弹出层](./popover.md)** - 点击弹出层
- **[TxNavBar 导航栏](./nav-bar.md)** - 移动端顶部导航栏
- **[TxTabBar 底部导航](./tab-bar.md)** - 移动端底部 TabBar
- **[TxDropdownMenu 下拉菜单](./dropdown-menu.md)** - 下拉菜单
- **[TxContextMenu 右键菜单](./context-menu.md)** - 右键菜单
- **[TxEmpty 空状态](./empty.md)** - 空状态占位

- **[TxLoadingState 加载态](./loading-state.md)** - 状态加载占位
- **[TxNoSelection 未选择](./no-selection.md)** - 详情面板未选择占位

## 快速开始

```bash
npm install @talex-touch/tuffex
```

```vue
<template>
  <TxButton type="primary">你好 TuffEx</TxButton>
  <TxTag label="新" color="var(--tx-color-success)" />
  <TxStatusBadge text="活跃" status="success" />
</template>
```
